## Little online shop

To run the application:
1) Clone the repository by link
   ```shell
    git clone https://github.com/NikBel3476/device-online-shop-server.git
    cd online-shop
   ```
2) Install dependencies
    ```shell
    npm i
    ```
   or
    ```shell
    npm install
    ```
3) Run the application
    ```shell
    npm run dev
    ```
   App will start on http://localhost:5000
